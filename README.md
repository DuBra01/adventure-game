# 🧭 Adventure Game

A text-based adventure game built in Python as part of a course project.  
Embark on a journey through a mysterious forest and a dark cave in search of a legendary treasure.

---

## 🎮 Features

- Interactive command-line gameplay
- Multiple decision paths (forest & cave)
- Win and lose conditions
- Restart functionality with personalized experience
- Flexible input handling (`1`, `forest`, `yes`, `y`, etc.)

---

## ▶️ How to Run

1. Clone the repository:

git clone https://github.com/DuBra01/adventure-game.git

2. Navigate to the project folder:

cd adventure-game

3. Run the game:

python3 adventure_game.py

---

## 🎮 How to Play

- Follow the instructions displayed in the terminal
- Enter your choices when prompted
- You can use:
  - Numbers → `1`, `2`, etc.
  - Words → `forest`, `cave`, `yes`, `no`

### 🧠 Game Logic

- Some choices help you progress
- Some choices lead to immediate failure
- Only the correct path leads to the treasure 🏆

---

## 🔁 Restarting the Game

After the game ends:

- Type `hi` → start a new adventure  
- Type `exit` → quit the game  

If you exit the game completely, you can restart anytime by running:

python3 adventure_game.py
